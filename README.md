### Instalação

Para instalá-las abra o terminal do seu computador, navegue até a pasta raiz do projeto e digite:

```
npm install
ou
yarn
```

Após a instalação das dependências, basta executar o seguinte comando:

* Desenvolvimento
```
> npm run dev
```
